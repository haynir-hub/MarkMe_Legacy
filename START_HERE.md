# 🚀 Quick Start - Two-Phase Tracking UI

## להתחיל מהר!

### Mac:
**קליק כפול על:**
```
launch_two_phase.command
```

### Windows:
**קליק כפול על:**
```
launch_two_phase.bat
```

---

## מה קורה אחרי הקליק?

1. חלון בחירת וידאו ייפתח
2. בחר וידאו (mp4, mov, avi, mkv)
3. המערכת הדו-שלבית תיפתח!

---

## שימוש מהיר:

### 1️⃣ הוסף שחקנים (2 דרכים):

**דרך א: זיהוי אוטומטי**
- לחץ "🔍 Detect"
- קליק על דמות בווידאו
- הכנס שם + בחר מרקר

**דרך ב: ציור ידני**
- לחץ "✏️ Draw"
- קליק וגרור bbox על הווידאו
- הכנס שם + בחר מרקר

### 2️⃣ התחל מעקב:
- וודא שיש ✓ ליד השחקן
- לחץ "▶️ Track"

### 3️⃣ בדוק תוצאות:
- גרף ירוק = טוב ✅
- גרף אדום = בעיה ⚠️
- קליק על הגרף = קפוץ לפריים

### 4️⃣ ייצוא:
- לחץ "✅ Continue to Export"

---

## 💡 טיפים:

- **חיסכון בזמן**: השתמש ב-"Start/End Frame" לעקוב רק בחלק מהסרטון
- **כפתורים**: ריחוף עם העכבר = הסברים מפורטים
- **מספר שחקנים**: הוסף כמה שרוצה, בחר מי לעקוב עם checkboxes
